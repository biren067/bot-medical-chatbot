# Repository: bot-medical-chatbot

# How to run?

### STEPS:

Clone the repository

```bash
git clone https://github.com/biren067/bot-medical-chatbot.git
```

## Condo environment

### step 01:

```bash
conda create -n medical-bot python=3.10 -y
```

### step 02:

```bash
conda activate medical-bot
```

### step 03:

```bash
pip install -r requirements.txt
```

## using python env

### step 01:

```bash
python -m venv .venv
```

### step 02:

```bash
cd .venv/Script
activate
```

### step 03:

```
cd ../..
pip install -r requirements.txt
```
